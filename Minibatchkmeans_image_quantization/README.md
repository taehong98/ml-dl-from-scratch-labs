# MinibatchKmeans image quantization

## What I did
- 이미지를 RGB로 로드하고 픽셀 값을 0~1로 정규화했다(실패 시 랜덤 이미지 생성).
- 이미지를 `(픽셀수, 3)` 형태로 펼친 뒤 `MiniBatchKMeans`로 대표 색상 `K`개를 학습했다.
- 각 픽셀을 가장 가까운 대표 색상으로 치환해 양자화(색상 수 축소) 이미지를 복원했다.
- 원본/양자화 이미지와 추출된 색상 팔레트를 시각화했다.

## What I tested
- K-means 기반 색상 양자화가 이미지의 색상을 `K`개 대표색으로 압축하는지
- `K` 값 변화에 따라 단순화 정도(정보 손실/원본 유사도)가 어떻게 달라지는지
- 대표 색상(centers)이 팔레트로 의미 있게 추출되는지
